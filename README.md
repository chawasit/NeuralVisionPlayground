# Neural Vision Playground
An Interactive Convolutional Neural Network For Kids

## Requirements
- python 3.5+
- tensorflow
- nodejs v8+

## How to use
- $ pip install -r requirements.txt
- frontend$ npm install
- frontend$ vue build src/main.js
- $ python server.py
- Access http://localhost:5000
